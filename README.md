# MockMaster

MockMaster is a full-stack, AI-powered mock interview application featuring a React frontend and a Python FastAPI backend built with the Google ADK. It utilizes three specialized agents—Scout, Interviewer, and Auditor—connected via REST and WebSocket APIs to provide seamless, real-time interactive technical interviews.

View your app in AI Studio: https://ai.studio/apps/8870c037-f15a-4d7d-abff-c7c7f6d01ea9

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
